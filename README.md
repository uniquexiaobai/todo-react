# Todo

使用 create-react-app 写 [TodoMVC](https://github.com/tastejs/todomvc)

## 预览
[http://uniquexiaobai.cn/todo-react/](http://uniquexiaobai.cn/todo-react/)

## 安装与运行

克隆项目

```
> $ git clone https://github.com/uniquexiaobai/todo-react.git
```

安装依赖
```
> $ cd todo-react && yarn
```

启动开发者环境
```
> $ yarn start
```

构建发布
```
> $ yarn run build
```
